# 리뷰 질문 템플릿
리뷰어가 바로 복사해 사용할 수 있는 질문 목록입니다.

## 핵심 질문
1. 경계: 해당 컨텍스트가 책임지는 용어가 두 컨텍스트에 동시에 등장하나요?
2. 책임: 애그리게잇 루트가 거절/승인 권한을 모두 가져야 하는 이유는 무엇인가요?
3. 트랜잭션: 상태 전이 중 롤백 시 어떤 이벤트가 캡처되나요?
4. 이벤트: 이벤트 미발행 시 어떤 정책이 무력화되나요?
5. 상태: `체크인` 없이 `체크아웃`이 가능하도록 열어둔 부분이 있나요?

## 사용법
- 정답 제시 금지. 각 질문을 코멘트에 붙여넣고, 필요 시 컨텍스트를 추가하여 토론을 유도하세요.
- 오프라인 토론용 질문은 "[오프라인]" 태그를 붙여 분류합니다.

## 체크리스트
- [ ] 위 질문 중 최소 2개 이상을 사용했다.
- [ ] 질문이 시나리오/테스트 결과를 참고하고 있다.
- [ ] 정책/불변조건 중 어느 쪽을 겨냥했는지 명시했다.
